# 🚀 Featured Repositories

### 🔐 Secure File Transfer System (SFTS)

A production-ready secure file transfer platform built with **Flask**, focused on secure authentication, encrypted file transfers, and enterprise-grade security practices.

✨ **Key Features**
- 🔑 Secure User Authentication & Authorization
- 🛡️ Multi-Factor Authentication (TOTP + Backup Codes)
- 🔒 End-to-End AES File Encryption
- 🔐 SHA-256 Integrity Verification
- 📁 Secure File Upload & Download
- 📊 Admin Dashboard & Security Logs
- ⚡ Session Management with Sliding Expiration
- 🚨 Brute Force Protection & Rate Limiting
- 📈 User & Security Analytics
- 🐳 Docker Ready Deployment

**Tech Stack:** Flask • Python • SQLite • HTML/CSS/JS • AES • SHA-256 • TOTP • Docker

⭐ **Repository:** https://github.com/SudharsanM7/SFTS
